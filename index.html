Introduction to JS

JavaScript is the world's most popular programming language. It is the programming language of the Web.
JavaScript is easy to learn.

First we will see how to run a JS code and display words on a screen called console.

console.log() prints the text characters in the paranthesis. Such text characters are called strings.

Strings are always enclosed in single qoutes or double qoutes. console.log() will display them the same way.
